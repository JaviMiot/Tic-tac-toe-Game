## Installation🚀

Clone repository

```
git clone https://github.com/JaviMiot/Tic-tac-toe-Game.git
```

Run project

```
npm run start
```

### Pre-requisites 📋

_What I need installs?_

- Node
- React

## Deploy 📦

This project was deploy in [netlify](https://www.netlify.com/)

```
npm run build
```
